OBM Controlsystem web application
Gregory Rozenbaum

## Как пользоваться?
Нужно установить Node.js (Npm) и gulp `npm install gulp -g`

Инсталлируем зависимости:
```
npm install
```

Сборка dev:
```
gulp
```

Сборка prod:
```
gulp prod
```
